Seu site de Jogos Arcade Gratuito!

A arcade-point ajuda a comunidade com jogos gratuitos e os desenvolvedores com o feedback da comunidade para seus primeiros jogos, assim podendo melhorar o nivel dos jogos arcade no cenario atual.

Sobre nós

Somos uma equipe de desenvolvedores Universitarios, que tivemos a brilhante ideia de remeter as memorias de jogos com aquela pegada arcade com isso criamos a primeira biblioteca de jogos arcade para comunidade e para os novos desenvolvedores de jogos.

Juntando o util ao agradavel,aprensentamos essa uma plataforma web com o intuito de fazer a comunidade voltar a jogar mais jogos arcade como antigamente =) .

Contudo sempre precisamos de ajuda para manter a plataforma para a comunidade então se puderem contribuir com algum valor seriamos gratos: CONTRIBUA
